# Instruções de Execução do Projeto RSA

Para executar o programa RSA desenvolvido em C#, é necessário ter o **.NET SDK** instalado no computador.

---

## 1. Instalar o .NET SDK

Baixe e instale o .NET SDK pelo site oficial:

https://dotnet.microsoft.com/download

Depois de instalar, abra o PowerShell e verifique se o `dotnet` está funcionando:

```powershell
dotnet --version
