# Trabalho-Seguranca
Trabalho realizado para a matéria de Segurança da Informação. 
O código pega um texto, criptografa com RSA, depois descriptografa e mostra o resultado passo a passo na tela.


Esse código é um programa que mostra como funciona a criptografia RSA. Ele:
- Pede ao usuário uma mensagem (ou assume "Olá Mundo!" se nada for digitado).
Gera duas chaves:
- Uma pública (para criptografar)
- Uma privada (para descriptografar)
- Converte a mensagem em números.
- Criptografa esses números usando a chave pública.
- Depois descriptografa usando a chave privada.
- Mostra tudo na tela: mensagem original, chave pública, chave privada, mensagem criptografada e a final recuperada.
